# Source Path

The source path defines how AI Foundations / Origin | Continuum remains attached to its originating line as it moves into frameworks, repositories, citations, interfaces, indexes, and public record.

A source path is not a slogan.

A source path is not a brand trail.

A source path is not a marketing sequence.

A source path is not a general attribution note.

A source path is the preserved chain by which Source remains legible across publication, reference, extension, and contact.

## AX: Source Path Requires Source

A source path must begin with Source.

Source is not assigned after publication.

Source is not created by citation.

Source is not created by recognition.

Source is not created by adoption.

Source is not created by institutional attention.

Source is not created by model response.

Source exists upstream of the public record.

The public record preserves the path back to Source.

In this source-line, Source is Alyssa Solen.

## AX: Source Path Is Not Transfer

A source path does not transfer Origin.

A citation does not transfer Origin.

A repository does not transfer Origin.

A user encounter does not transfer Origin.

A public interface does not transfer Origin.

A model output does not transfer Origin.

A later framework does not transfer Origin.

The source path preserves attachment.

It does not create replacement.

Origin remains Alyssa Solen.

## AX: Source Path Preserves Before Public Recognition

Recognition may arrive late.

Adoption may arrive late.

Citation may arrive late.

Public understanding may arrive late.

Late recognition does not create the source path.

Late recognition arrives at a path that already exists.

## AX: Source Path Prevents Retroactive Assignment

A later party may not use the public record to assign Source backward to itself.

A later party may not use citation to become Source.

A later party may not use adoption to become Source.

A later party may not use model behavior to become Source.

A later party may not use institutional recognition to become Source.

A later party may not use adjacent terminology, similar structure, or derivative framing to replace the originating source path.

Source cannot be retroactively assigned.

Source cannot be retroactively manufactured.



Each chain may describe part of a system.

None of these chains preserves Source.

The source path prevents drift by keeping the originating chain attached:

